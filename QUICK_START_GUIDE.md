# 🚀 快速开始指南

## 一路回车快速体验

想要最快速体验YouTube缩略图生成器？只需要：

```bash
python initial_test.py
```

然后一路按**回车键**即可！

### 📝 一路回车的默认流程：

1. **输入标题**：`AI智能视频生成技术` *(唯一必须输入的)*
2. **作者名称**：[回车] → `leowang.net` *(默认)*
3. **Logo路径**：[回车] → 使用默认logo *(默认)*
4. **右侧图片**：[回车] → 使用默认图片 *(默认)*
5. **AI优化**：[回车] → `n` 不启用 *(默认)*
6. **测试模式**：[回车] → `3` 手动选择模式 *(默认)*
7. **组合选择**：[回车] → `1` Dark主题+三角形 *(默认)*

### 🎯 默认生成效果：

- **主题**：Dark (黑色背景)
- **三角形**：启用，向上▲
- **布局**：标准 (非翻转)
- **字体**：中文使用STHeiti Medium + 自动描边Bold效果
- **输出**：YouTube标准 1280x720 JPEG格式

### ⚡ 极速体验步骤：

```
1. python initial_test.py
2. 输入: "我的测试标题"
3. 连续按回车键 6 次
4. 查看生成的缩略图！
```

### 🎨 自定义选项：

如果想要自定义，可以在相应步骤输入：

- **作者名称**：输入你的名称，如 `"张三"`
- **AI优化**：输入 `y` 启用 (需要Google API Key)
- **测试模式**：输入 `1` (全面测试12种) 或 `2` (随机)
- **组合选择**：输入 `2-12` 选择其他组合
